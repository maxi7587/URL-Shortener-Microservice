To shorten a new URL use the "/new" path before the URL:

https://url-shortener-app.glitch.me/new/https://www.google.com

You'll get a response like:

{"original_url":"https://www.google.com","short_url":"https://url-shortener-app.glitch.me/599a537c8dc8d02e9734007d"}

You can use the "short_url" as you like, it will redirect automatically to the "original_url". Test it in your browser.

Enjoy!!!
